# Project Documentation

## Overview
This project is a flight tracking application that allows users to search for, view, and manage flight information.

## Features
- Search for flights
- View flight details
- Manage flight bookings

## Technology Stack
- React
- Node.js
- Express

## Installation
1. Clone the repository: `git clone https://github.com/denniscihan1-lab/FLIGHT-TRACK-FRONTIER.git`
2. Navigate to the project directory: `cd FLIGHT-TRACK-FRONTIER`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Usage
- Visit the application in your browser: `http://localhost:3000`
